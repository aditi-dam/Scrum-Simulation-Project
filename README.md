# Scrum-Simulation-Project
CS 555 Scrum Simulation Project. A to-do list web app that uses AI to categorize tasks.

# To-Do List App

## Features
- User authentication (register, login, logout)
- Continue as guest option
- Add tasks
- Delete tasks
- Mark tasks complete
- Task categories (work, school, home)
- Due dates
- Task color customization
- User-specific task lists
- Home page (active tasks)
- Completed tasks page
- Add task page

## How to Run

### Server
cd server
npm install
npm run dev or npm start

## Server runs on
http://localhost:5000

## Open Webapp on
http://localhost:5000/auth/login
